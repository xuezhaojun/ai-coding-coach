# 丢失的数字

- **难度**: 简单
- **分类**: 位运算
- **考点**: 数组, 位运算, 数学, 异或
- **链接**: [NeetCode](https://neetcode.io/problems/missing-number) | [力扣 268](https://leetcode.cn/problems/missing-number/)

## 题目描述

给定一个包含 `[0, n]` 中 `n` 个不同数字的数组 `nums`，返回该范围内唯一缺少的数字。

你应该实现一个仅使用 `O(1)` 额外空间且时间复杂度为 `O(n)` 的解法。

## 示例

**示例 1:**

```
输入: nums = [3,0,1]
输出: 2
解释: n = 3，因为有 3 个数字，所以所有数字在 [0,3] 范围内。2 是缺失的数字。
```

**示例 2:**

```
输入: nums = [0,1]
输出: 2
解释: n = 2，因为有 2 个数字，所以所有数字在 [0,2] 范围内。2 是缺失的数字。
```

**示例 3:**

```
输入: nums = [9,6,4,2,3,5,7,0,1]
输出: 8
解释: n = 9，因为有 9 个数字，所以所有数字在 [0,9] 范围内。8 是缺失的数字。
```

## 约束条件

- `n == nums.length`
- `1 <= n <= 10^4`
- `0 <= nums[i] <= n`
- `nums` 中的所有数字互不相同

## 函数签名

```go
func missingNumber(nums []int) int
```
