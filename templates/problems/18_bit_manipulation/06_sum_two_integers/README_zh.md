# 两整数之和

- **难度**: 中等
- **分类**: 位运算
- **考点**: 位运算, 数学
- **链接**: [NeetCode](https://neetcode.io/problems/sum-of-two-integers) | [力扣 371](https://leetcode.cn/problems/sum-of-two-integers/)

## 题目描述

给你两个整数 `a` 和 `b`，不使用运算符 `+` 和 `-`，计算并返回两整数之和。你必须使用位运算来计算。

## 示例

**示例 1:**

```
输入: a = 1, b = 2
输出: 3
解释: 1 + 2 = 3，使用位运算计算。
```

**示例 2:**

```
输入: a = 2, b = -1
输出: 1
解释: 2 + (-1) = 1，使用位运算计算。
```

**示例 3:**

```
输入: a = 0, b = 0
输出: 0
解释: 0 + 0 = 0。
```

## 约束条件

- `-1000 <= a, b <= 1000`

## 函数签名

```go
func getSum(a int, b int) int
```
