# 只出现一次的数字

- **难度**: 简单
- **分类**: 位运算
- **考点**: 数组, 位运算, 异或
- **链接**: [NeetCode](https://neetcode.io/problems/single-number) | [力扣 136](https://leetcode.cn/problems/single-number/)

## 题目描述

给你一个非空整数数组 `nums`，除了某个元素只出现一次以外，其余每个元素均出现两次。找出那个只出现了一次的元素。

你必须设计一个线性时间复杂度的算法，且只使用常量级别的额外空间来解决此问题。

## 示例

**示例 1:**

```
输入: nums = [2,2,1]
输出: 1
解释: 1 是唯一只出现一次的元素。
```

**示例 2:**

```
输入: nums = [4,1,2,1,2]
输出: 4
解释: 4 是唯一只出现一次的元素；1 和 2 各出现两次。
```

**示例 3:**

```
输入: nums = [1]
输出: 1
解释: 数组中只有一个元素，即为只出现一次的数字。
```

## 约束条件

- `1 <= nums.length <= 3 * 10^4`
- `-3 * 10^4 <= nums[i] <= 3 * 10^4`
- 数组中每个元素恰好出现两次，只有一个元素出现一次

## 函数签名

```go
func singleNumber(nums []int) int
```
