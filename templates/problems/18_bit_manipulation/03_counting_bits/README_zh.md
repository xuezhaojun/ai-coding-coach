# 比特位计数

- **难度**: 简单
- **分类**: 位运算
- **考点**: 位运算, 动态规划
- **链接**: [NeetCode](https://neetcode.io/problems/counting-bits) | [力扣 338](https://leetcode.cn/problems/counting-bits/)

## 题目描述

给你一个整数 `n`，返回一个长度为 `n + 1` 的数组 `ans`，其中 `ans[i]` 是 `i` 的二进制表示中 `1` 的个数（`0 <= i <= n`）。

你应该找到一个 `O(n)` 时间复杂度的解法，不要对每个数使用内置的 popcount 函数。

## 示例

**示例 1:**

```
输入: n = 2
输出: [0,1,1]
解释: 0 -> 0（零个 1），1 -> 1（一个 1），2 -> 10（一个 1）。
```

**示例 2:**

```
输入: n = 5
输出: [0,1,1,2,1,2]
解释: 0->0, 1->1, 2->10, 3->11, 4->100, 5->101。
```

**示例 3:**

```
输入: n = 0
输出: [0]
解释: 只有 0 本身，有零个 1 位。
```

## 约束条件

- `0 <= n <= 10^5`

## 函数签名

```go
func countBits(n int) []int
```
