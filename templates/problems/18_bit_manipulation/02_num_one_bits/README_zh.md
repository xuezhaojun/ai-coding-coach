# 位1的个数

- **难度**: 简单
- **分类**: 位运算
- **考点**: 位运算
- **链接**: [NeetCode](https://neetcode.io/problems/number-of-one-bits) | [力扣 191](https://leetcode.cn/problems/number-of-1-bits/)

## 题目描述

编写一个函数，输入一个正整数的二进制表示，返回其中设置位（1 位）的个数，也称为汉明重量。

## 示例

**示例 1:**

```
输入: n = 11（二进制: 00000000000000000000000000001011）
输出: 3
解释: 二进制表示中有三个 1 位。
```

**示例 2:**

```
输入: n = 128（二进制: 00000000000000000000000010000000）
输出: 1
解释: 二进制表示中有一个 1 位。
```

**示例 3:**

```
输入: n = 4294967293（二进制: 11111111111111111111111111111101）
输出: 31
解释: 二进制表示中有三十一个 1 位。
```

## 约束条件

- 输入是一个 32 位无符号整数

## 函数签名

```go
func hammingWeight(n uint32) int
```
