# 颠倒二进制位

- **难度**: 简单
- **分类**: 位运算
- **考点**: 位运算
- **链接**: [NeetCode](https://neetcode.io/problems/reverse-bits) | [力扣 190](https://leetcode.cn/problems/reverse-bits/)

## 题目描述

颠倒给定的 32 位无符号整数的二进制位。例如，如果输入的二进制表示为 `00000010100101000001111010011100`，输出应为 `00111001011110000010100101000000`，对应无符号整数 `964176192`。

## 示例

**示例 1:**

```
输入: n = 00000010100101000001111010011100
输出: 964176192 (00111001011110000010100101000000)
解释: 输入表示无符号整数 43261596，颠倒其二进制位后得到 964176192。
```

**示例 2:**

```
输入: n = 11111111111111111111111111111101
输出: 3221225471 (10111111111111111111111111111111)
解释: 输入表示无符号整数 4294967293，颠倒其二进制位后得到 3221225471。
```

**示例 3:**

```
输入: n = 1
输出: 2147483648 (10000000000000000000000000000000)
解释: 颠倒 1 的二进制位后，唯一的设置位移到第 31 位。
```

## 约束条件

- 输入必须是长度为 `32` 的二进制串

## 函数签名

```go
func reverseBits(num uint32) uint32
```
