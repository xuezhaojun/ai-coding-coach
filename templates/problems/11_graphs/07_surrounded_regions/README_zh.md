# 被围绕的区域

- **难度**: 中等
- **分类**: 图
- **考点**: 深度优先搜索, 广度优先搜索, 矩阵, 洪水填充
- **链接**: [NeetCode](https://neetcode.io/problems/surrounded-regions) | [力扣 130](https://leetcode.cn/problems/surrounded-regions/)

## 题目描述

给你一个 `m x n` 的矩阵 `board`，由若干字符 `'X'` 和 `'O'` 组成，找到所有被 `'X'` 围绕的区域，并将这些区域里所有的 `'O'` 用 `'X'` 填充。

被围绕的区间不会存在于边界上。也就是说，任何边界上的 `'O'` 都不会被填充为 `'X'`。任何不在边界上且不与边界上的 `'O'` 相连的 `'O'` 最终都会被填充为 `'X'`。

## 示例

**示例 1:**

```
输入: board = [
  ["X","X","X","X"],
  ["X","O","O","X"],
  ["X","X","O","X"],
  ["X","O","X","X"]
]
输出: [
  ["X","X","X","X"],
  ["X","X","X","X"],
  ["X","X","X","X"],
  ["X","O","X","X"]
]
解释: 左下角的 'O' 位于边界上，不会被填充。其余的 'O' 被围绕，被填充为 'X'。
```

**示例 2:**

```
输入: board = [["X"]]
输出: [["X"]]
```

**示例 3:**

```
输入: board = [
  ["X","O","X"],
  ["X","O","X"],
  ["X","X","X"]
]
输出: [
  ["X","O","X"],
  ["X","O","X"],
  ["X","X","X"]
]
解释: 'O' 单元格连接到顶部边界，因此不被围绕。
```

## 约束条件

- `m == board.length`
- `n == board[i].length`
- `1 <= m, n <= 200`
- `board[i][j]` 为 `'X'` 或 `'O'`。
