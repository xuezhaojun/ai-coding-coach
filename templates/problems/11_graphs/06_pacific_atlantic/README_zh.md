# 太平洋大西洋水流问题

- **难度**: 中等
- **分类**: 图
- **考点**: 深度优先搜索, 广度优先搜索, 矩阵
- **链接**: [NeetCode](https://neetcode.io/problems/pacific-atlantic-water-flow) | [力扣 417](https://leetcode.cn/problems/pacific-atlantic-water-flow/)

## 题目描述

有一个 `m x n` 的矩形岛屿，与太平洋和大西洋相邻。太平洋处于岛屿的左边界和上边界，大西洋处于岛屿的右边界和下边界。

岛屿被分割为一个由方形单元格组成的网格。给你一个 `m x n` 的整数矩阵 `heights`，其中 `heights[r][c]` 表示坐标 `(r, c)` 上单元格高于海平面的高度。

岛屿上雨水较多，如果相邻单元格的高度小于或等于当前单元格的高度，雨水可以直接向北、南、东、西流向相邻单元格。水可以从海洋附近的任何单元格流入海洋。

返回一个二维列表，其中 `result[i] = [ri, ci]` 表示雨水从单元格 `(ri, ci)` 流动既可以流到太平洋，也可以流到大西洋。

## 示例

**示例 1:**

```
输入: heights = [[1,2,2,3,5],[3,2,3,4,4],[2,4,5,3,1],[6,7,1,4,5],[5,1,1,2,4]]
输出: [[0,4],[1,3],[1,4],[2,2],[3,0],[3,1],[4,0]]
```

**示例 2:**

```
输入: heights = [[1]]
输出: [[0,0]]
```

**示例 3:**

```
输入: heights = [[1,1],[1,1]]
输出: [[0,0],[0,1],[1,0],[1,1]]
解释: 在平坦的网格上，每个单元格的水都可以流到两个大洋。
```

## 约束条件

- `m == heights.length`
- `n == heights[r].length`
- `1 <= m, n <= 200`
- `0 <= heights[r][c] <= 10^5`

## 函数签名

```go
func pacificAtlantic(heights [][]int) [][]int
```
