# 克隆图

- **难度**: 中等
- **分类**: 图
- **考点**: 深度优先搜索, 广度优先搜索, 哈希表, 图遍历
- **链接**: [NeetCode](https://neetcode.io/problems/clone-graph) | [力扣 133](https://leetcode.cn/problems/clone-graph/)

## 题目描述

给你无向连通图中一个节点的引用，请你返回该图的深拷贝（克隆）。图中的每个节点都包含它的值 `val`（`int`）和其邻居的列表（`list[Node]`）。

测试用例使用邻接列表表示图。每个节点的值与其索引相同（1 索引）。返回的克隆图必须由全新的节点对象组成，不能与原图的节点共享指针。

## 示例

**示例 1:**

```
输入: adjList = [[2,4],[1,3],[2,4],[1,3]]
输出: [[2,4],[1,3],[2,4],[1,3]]
解释: 图中有 4 个节点。节点 1 连接到节点 2 和 4，节点 2 连接到节点 1 和 3。克隆图是使用新节点对象的深拷贝。
```

**示例 2:**

```
输入: adjList = [[]]
输出: [[]]
解释: 只有一个节点且没有邻居。
```

**示例 3:**

```
输入: adjList = []
输出: []
解释: 图为空（输入为 null）。
```

## 约束条件

- 图中的节点数在 `[0, 100]` 范围内。
- `1 <= Node.val <= 100`
- 每个节点的值 `Node.val` 都是唯一的。
- 图中没有重复的边，也没有自环。
- 图是连通的，从给定节点出发可以访问所有节点。

## 函数签名

```go
type Node struct {
    Val       int
    Neighbors []*Node
}

func cloneGraph(node *Node) *Node
```
