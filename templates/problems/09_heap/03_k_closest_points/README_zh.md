# 最接近原点的 K 个点

- **难度**: 中等
- **分类**: 堆
- **考点**: 最大堆, 优先队列, 几何
- **链接**: [NeetCode](https://neetcode.io/problems/k-closest-points-to-origin) | [力扣 973](https://leetcode.cn/problems/k-closest-points-to-origin/)

## 题目描述

给定一个点数组 `points`，其中 `points[i] = [xi, yi]` 表示 X-Y 平面上的一个点，以及一个整数 `k`，返回离原点 `(0, 0)` 最近的 `k` 个点。两点之间的距离为欧几里得距离：`sqrt(xi^2 + yi^2)`。可以按任意顺序返回答案，答案保证唯一（除了返回的顺序可能不同）。

## 示例

**示例 1:**

```
输入: points = [[1,3],[-2,2]], k = 1
输出: [[-2,2]]
解释: (1,3) 到原点的距离 = sqrt(10)。(-2,2) 到原点的距离 = sqrt(8)。因为 sqrt(8) < sqrt(10)，所以 (-2,2) 最近。
```

**示例 2:**

```
输入: points = [[3,3],[5,-1],[-2,4]], k = 2
输出: [[3,3],[-2,4]]
解释: 距离分别为 sqrt(18)、sqrt(26)、sqrt(20)。最近的两个点是 (3,3) 和 (-2,4)。
```

**示例 3:**

```
输入: points = [[0,0],[1,1],[2,2]], k = 1
输出: [[0,0]]
解释: 原点本身距离为 0，是最小的。
```

## 约束条件

- `1 <= k <= points.length <= 10^4`
- `-10^4 <= xi, yi <= 10^4`

## 函数签名

```go
func kClosest(points [][]int, k int) [][]int
```
