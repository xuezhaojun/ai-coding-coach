# 最后一块石头的重量

- **难度**: 简单
- **分类**: 堆
- **考点**: 最大堆, 优先队列, 模拟
- **链接**: [NeetCode](https://neetcode.io/problems/last-stone-weight) | [力扣 1046](https://leetcode.cn/problems/last-stone-weight/)

## 题目描述

有一堆石头，用整数数组 `stones` 表示，其中 `stones[i]` 是第 `i` 块石头的重量。每一回合，从中选出最重的两块石头，然后将它们一起粉碎。假设石头的重量分别为 `x` 和 `y`，且 `x <= y`。如果 `x == y`，那么两块石头都会被完全粉碎。如果 `x != y`，那么重量为 `x` 的石头将被完全粉碎，而重量为 `y` 的石头新重量为 `y - x`。最后，最多只会剩下一块石头。返回此石头最小的可能重量。如果没有石头剩下，就返回 `0`。

## 示例

**示例 1:**

```
输入: stones = [2,7,4,1,8,1]
输出: 1
解释: 粉碎 7 和 8，剩余 1。粉碎 4 和 2，剩余 2。粉碎 2 和 1，剩余 1。粉碎 1 和 1，两者都被摧毁。最终剩余一块重量为 1 的石头。
```

**示例 2:**

```
输入: stones = [3,3]
输出: 0
解释: 两块石头重量相同，全部被粉碎。没有石头剩余。
```

**示例 3:**

```
输入: stones = [1]
输出: 1
解释: 只有一块石头，它就是最后剩余的石头。
```

## 约束条件

- `1 <= stones.length <= 30`
- `1 <= stones[i] <= 1000`

## 函数签名

```go
func lastStoneWeight(stones []int) int
```
