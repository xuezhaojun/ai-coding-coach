# 任务调度器

- **难度**: 中等
- **分类**: 堆
- **考点**: 最大堆, 贪心, 调度
- **链接**: [NeetCode](https://neetcode.io/problems/task-scheduling) | [力扣 621](https://leetcode.cn/problems/task-scheduler/)

## 题目描述

给你一个用字符（A 到 Z）表示的 CPU 任务数组和一个冷却间隔 `n`。每个周期可以完成一个任务。任务可以按任意顺序完成，但有一个约束：相同的任务之间必须间隔至少 `n` 个周期。如果在冷却间隔期间没有可调度的任务，CPU 将保持空闲。返回 CPU 完成所有任务所需的最少周期数。

## 示例

**示例 1:**

```
输入: tasks = ['A','A','A','B','B','B'], n = 2
输出: 8
解释: 一种可能的调度方案：A B 空闲 A B 空闲 A B。任务 A 和 B 各出现 3 次，且必须间隔至少 2 个周期。
```

**示例 2:**

```
输入: tasks = ['A','A','A','B','B','B'], n = 0
输出: 6
解释: 没有冷却时间，任务可以连续执行：ABABAB 或任意顺序。共 6 个周期。
```

**示例 3:**

```
输入: tasks = ['A','A','A','A','A','A','B','C','D','E','F','G'], n = 2
输出: 16
解释: 一种可能的调度方案：A B C A D E A F G A 空闲 空闲 A 空闲 空闲 A。任务 A 出现 6 次，主导了调度。
```

## 约束条件

- `1 <= tasks.length <= 10^4`
- `tasks[i]` 是大写英文字母。
- `0 <= n <= 100`

## 函数签名

```go
func leastInterval(tasks []byte, n int) int
```
