# 爬楼梯

- **难度**: 简单
- **分类**: 一维动态规划
- **考点**: 动态规划, 斐波那契数列
- **链接**: [NeetCode](https://neetcode.io/problems/climbing-stairs) | [力扣 70](https://leetcode.cn/problems/climbing-stairs/)

## 题目描述

假设你正在爬楼梯，需要 `n` 阶你才能到达楼顶。每次你可以爬 1 或 2 个台阶。请问有多少种不同的方法可以爬到楼顶？

## 示例

**示例 1:**

```
输入: n = 2
输出: 2
解释: 有两种方法可以爬到楼顶：1+1 和 2。
```

**示例 2:**

```
输入: n = 3
输出: 3
解释: 有三种方法：1+1+1、1+2 和 2+1。
```

**示例 3:**

```
输入: n = 5
输出: 8
```

## 约束条件

- `1 <= n <= 45`

## 函数签名

```go
func climbStairs(n int) int
```
