# 打家劫舍

- **难度**: 中等
- **分类**: 一维动态规划
- **考点**: 动态规划, 数组
- **链接**: [NeetCode](https://neetcode.io/problems/house-robber) | [力扣 198](https://leetcode.cn/problems/house-robber/)

## 题目描述

你是一个专业的小偷，计划偷窃沿街的房屋。每间房内都藏有一定的现金。相邻的房屋装有相互连通的防盗系统，如果两间相邻的房屋在同一晚上被小偷闯入，系统会自动报警。给定一个代表每个房屋存放金额的非负整数数组 `nums`，请计算在不触动警报装置的情况下，一夜之内能够偷窃到的最高金额。

## 示例

**示例 1:**

```
输入: nums = [1,2,3,1]
输出: 4
解释: 偷窃第 0 号房屋（金额 = 1）和第 2 号房屋（金额 = 3）。总金额 = 1 + 3 = 4。
```

**示例 2:**

```
输入: nums = [2,7,9,3,1]
输出: 12
解释: 偷窃第 0 号房屋（金额 = 2）、第 2 号房屋（金额 = 9）和第 4 号房屋（金额 = 1）。总金额 = 2 + 9 + 1 = 12。
```

**示例 3:**

```
输入: nums = [100,1,100,1,100]
输出: 300
```

## 约束条件

- `0 <= nums.length <= 100`
- `0 <= nums[i] <= 400`

## 函数签名

```go
func rob(nums []int) int
```
