# 打家劫舍 II

- **难度**: 中等
- **分类**: 一维动态规划
- **考点**: 动态规划, 数组
- **链接**: [NeetCode](https://neetcode.io/problems/house-robber-ii) | [力扣 213](https://leetcode.cn/problems/house-robber-ii/)

## 题目描述

你是一个专业的小偷，计划偷窃沿街的房屋，所有房屋围成一圈。相邻的房屋装有相互连通的防盗系统，如果两间相邻的房屋在同一晚上被小偷闯入，系统会自动报警。注意第一间房屋和最后一间房屋也是相邻的。给定一个代表每个房屋存放金额的非负整数数组 `nums`，请计算在不触动警报装置的情况下，一夜之内能够偷窃到的最高金额。

## 示例

**示例 1:**

```
输入: nums = [2,3,2]
输出: 3
解释: 不能同时偷窃第 0 号和第 2 号房屋，因为它们在圆环中相邻。偷窃第 1 号房屋（金额 = 3）。
```

**示例 2:**

```
输入: nums = [1,2,3,1]
输出: 4
解释: 偷窃第 0 号房屋（金额 = 1）和第 2 号房屋（金额 = 3）。总金额 = 4。
```

**示例 3:**

```
输入: nums = [1,2,3]
输出: 3
```

## 约束条件

- `1 <= nums.length <= 100`
- `0 <= nums[i] <= 1000`

## 函数签名

```go
func robII(nums []int) int
```
