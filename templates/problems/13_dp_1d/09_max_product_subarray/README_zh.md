# 乘积最大子数组

- **难度**: 中等
- **分类**: 一维动态规划
- **考点**: 动态规划, 数组
- **链接**: [NeetCode](https://neetcode.io/problems/maximum-product-subarray) | [力扣 152](https://leetcode.cn/problems/maximum-product-subarray/)

## 题目描述

给你一个整数数组 `nums`，请你找出数组中乘积最大的非空连续子数组，并返回该子数组所对应的乘积。数组中可能包含正数、负数和零。测试用例的答案保证是一个 32 位整数。

## 示例

**示例 1:**

```
输入: nums = [2,3,-2,4]
输出: 6
解释: 子数组 [2,3] 的乘积最大，为 6。
```

**示例 2:**

```
输入: nums = [-2,0,-1]
输出: 0
解释: 结果不能为 2，因为 [-2,-1] 不是连续子数组。
```

**示例 3:**

```
输入: nums = [-2,-3]
输出: 6
解释: 子数组 [-2,-3] 的乘积为 6。
```

## 约束条件

- `1 <= nums.length <= 2 * 10^4`
- `-10 <= nums[i] <= 10`
- `nums` 的任何子数组的乘积都保证是一个 32 位整数。

## 函数签名

```go
func maxProduct(nums []int) int
```
