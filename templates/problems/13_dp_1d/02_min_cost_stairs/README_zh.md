# 使用最小花费爬楼梯

- **难度**: 简单
- **分类**: 一维动态规划
- **考点**: 动态规划, 数组
- **链接**: [NeetCode](https://neetcode.io/problems/min-cost-climbing-stairs) | [力扣 746](https://leetcode.cn/problems/min-cost-climbing-stairs/)

## 题目描述

给你一个整数数组 `cost`，其中 `cost[i]` 是从楼梯第 `i` 个台阶向上爬需要支付的费用。一旦你支付此费用，即可选择向上爬一个或者两个台阶。你可以选择从下标为 0 或下标为 1 的台阶开始爬楼梯。请返回达到楼梯顶部（超过最后一个台阶）的最低花费。

## 示例

**示例 1:**

```
输入: cost = [10,15,20]
输出: 15
解释: 从下标 1 开始，支付 15，爬两个台阶即可到达楼顶。
```

**示例 2:**

```
输入: cost = [1,100,1,1,1,100,1,1,100,1]
输出: 6
解释: 从下标 0 开始，依次经过下标 0、2、4、6、7、9 的台阶，总花费为 6。
```

## 约束条件

- `2 <= cost.length <= 1000`
- `0 <= cost[i] <= 999`

## 函数签名

```go
func minCostClimbingStairs(cost []int) int
```
