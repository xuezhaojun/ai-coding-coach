# 目标和

- **难度**: 中等
- **分类**: 二维动态规划
- **考点**: 动态规划, 0/1 背包
- **链接**: [NeetCode](https://neetcode.io/problems/target-sum) | [力扣 494](https://leetcode.cn/problems/target-sum/)

## 题目描述

给你一个整数数组 `nums` 和一个整数 `target`。向数组中的每个整数前添加 '+' 或 '-'，然后串联起所有整数，可以构造一个表达式。返回可以通过上述方法构造的、运算结果等于 `target` 的不同表达式的数目。

## 示例

**示例 1:**

```
输入: nums = [1,1,1,1,1], target = 3
输出: 5
解释: 一共有 5 种方法使最终目标和为 3：-1+1+1+1+1=3，+1-1+1+1+1=3，+1+1-1+1+1=3，+1+1+1-1+1=3，+1+1+1+1-1=3。
```

**示例 2:**

```
输入: nums = [1], target = 1
输出: 1
```

**示例 3:**

```
输入: nums = [1], target = 2
输出: 0
解释: 无法通过单个值为 1 的元素达到目标值 2。
```

## 约束条件

- `1 <= nums.length <= 20`
- `0 <= nums[i] <= 1000`
- `0 <= sum(nums[i]) <= 1000`
- `-1000 <= target <= 1000`

## 函数签名

```go
func findTargetSumWays(nums []int, target int) int
```
