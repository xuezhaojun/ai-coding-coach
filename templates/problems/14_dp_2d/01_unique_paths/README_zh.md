# 不同路径

- **难度**: 中等
- **分类**: 二维动态规划
- **考点**: 动态规划, 数学, 组合数学
- **链接**: [NeetCode](https://neetcode.io/problems/unique-paths) | [力扣 62](https://leetcode.cn/problems/unique-paths/)

## 题目描述

一个机器人位于一个 `m x n` 网格的左上角（起始点在 `grid[0][0]`）。机器人每次只能向下或者向右移动一步。机器人试图达到网格的右下角（即 `grid[m-1][n-1]`）。给定两个整数 `m` 和 `n`，返回机器人从左上角到右下角的所有不同路径的数量。

![unique-paths](assets/robot_maze.png)

## 示例

**示例 1:**

```
输入: m = 3, n = 7
输出: 28
```

**示例 2:**

```
输入: m = 3, n = 2
输出: 3
解释: 从左上角开始，共有 3 条路径可以到达右下角：向右 -> 向下 -> 向下，向下 -> 向下 -> 向右，向下 -> 向右 -> 向下。
```

**示例 3:**

```
输入: m = 1, n = 1
输出: 1
```

## 约束条件

- `1 <= m, n <= 100`

## 函数签名

```go
func uniquePaths(m int, n int) int
```
