# 正则表达式匹配

- **难度**: 困难
- **分类**: 二维动态规划
- **考点**: 动态规划, 字符串, 递归
- **链接**: [NeetCode](https://neetcode.io/problems/regular-expression-matching) | [力扣 10](https://leetcode.cn/problems/regular-expression-matching/)

## 题目描述

给你一个字符串 `s` 和一个字符规律 `p`，请你来实现一个支持 `.` 和 `*` 的正则表达式匹配。`.` 匹配任意单个字符，`*` 匹配零个或多个前面的那一个元素。所谓匹配，是要涵盖整个字符串 `s`，而不是部分字符串。

## 示例

**示例 1:**

```
输入: s = "aa", p = "a"
输出: false
解释: "a" 无法匹配 "aa" 整个字符串。
```

**示例 2:**

```
输入: s = "aa", p = "a*"
输出: true
解释: '*' 代表可以匹配零个或多个前面的元素 'a'。因此，重复 'a' 两次即可匹配 "aa"。
```

**示例 3:**

```
输入: s = "ab", p = ".*"
输出: true
解释: ".*" 表示可匹配零个或多个任意字符，因此可以匹配 "ab"。
```

## 约束条件

- `1 <= s.length <= 20`
- `1 <= p.length <= 20`
- `s` 只包含小写英文字母。
- `p` 只包含小写英文字母、`.` 和 `*`。
- 保证每次出现 `*` 时，前面都有一个有效的字符与之匹配。

## 函数签名

```go
func isMatch(s string, p string) bool
```
