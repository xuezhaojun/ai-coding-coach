# 平衡二叉树

- **难度**: 简单
- **分类**: 树
- **考点**: 二叉树, 深度优先搜索, 递归
- **链接**: [NeetCode](https://neetcode.io/problems/balanced-binary-tree) | [力扣 110](https://leetcode.cn/problems/balanced-binary-tree/)

## 题目描述

给定一棵二叉树，判断它是否是高度平衡的。一棵高度平衡的二叉树定义为：树中每个节点的左右两个子树的高度差的绝对值不超过 1。

## 示例

**示例 1:**

```
输入: root = [3,9,20,null,null,15,7]
输出: true
解释: 左子树高度为 1，右子树高度为 2。差值为 1，每个节点都满足平衡条件。
```

**示例 2:**

```
输入: root = [1,2,2,3,3,null,null,4,4]
输出: false
解释: 根节点的左子树高度为 3，右子树高度为 1。差值为 2，因此树不平衡。
```

**示例 3:**

```
输入: root = []
输出: true
解释: 空树被认为是平衡的。
```

## 约束条件

- 树中节点数目范围在 `[0, 5000]` 内。
- `-10^4 <= Node.val <= 10^4`

## 函数签名

```go
func isBalanced(root *TreeNode) bool
```
