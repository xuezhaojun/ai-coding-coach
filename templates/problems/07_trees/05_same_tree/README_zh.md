# 相同的树

- **难度**: 简单
- **分类**: 树
- **考点**: 二叉树, 深度优先搜索, 递归
- **链接**: [NeetCode](https://neetcode.io/problems/same-binary-tree) | [力扣 100](https://leetcode.cn/problems/same-tree/)

## 题目描述

给定两棵二叉树的根节点 `p` 和 `q`，编写一个函数来检验它们是否相同。如果两棵树在结构上完全相同，并且每个对应位置的节点值都相等，则认为这两棵树是相同的。

## 示例

**示例 1:**

```
输入: p = [1,2,3], q = [1,2,3]
输出: true
解释: 两棵树具有相同的结构和相同的节点值。
```

**示例 2:**

```
输入: p = [1,2], q = [1,null,2]
输出: false
解释: 第一棵树的 2 是左子节点，但第二棵树的 2 是右子节点。结构不同。
```

**示例 3:**

```
输入: p = [1,2,1], q = [1,1,2]
输出: false
解释: 两棵树结构相同但对应位置的值不同。
```

## 约束条件

- 两棵树中的节点数目范围在 `[0, 100]` 内。
- `-10^4 <= Node.val <= 10^4`

## 函数签名

```go
func isSameTree(p *TreeNode, q *TreeNode) bool
```
