# 二叉树的右视图

- **难度**: 中等
- **分类**: 树
- **考点**: 二叉树, 广度优先搜索, 队列
- **链接**: [NeetCode](https://neetcode.io/problems/binary-tree-right-side-view) | [力扣 199](https://leetcode.cn/problems/binary-tree-right-side-view/)

## 题目描述

给定一棵二叉树的根节点，想象自己站在它的右侧，按照从顶部到底部的顺序，返回从右侧所能看到的节点值。对于树的每一层，只有最右边的节点从右侧可见。

## 示例

**示例 1:**

```
输入: root = [1,2,3,null,5,null,4]
输出: [1,3,4]
解释: 第 0 层看到节点 1，第 1 层看到节点 3（最右边），第 2 层看到节点 4（最右边）。
```

**示例 2:**

```
输入: root = [1,null,3]
输出: [1,3]
解释: 第 0 层看到节点 1，第 1 层看到节点 3。
```

**示例 3:**

```
输入: root = [1,2,3,4]
输出: [1,3,4]
解释: 第 0 层看到 1，第 1 层节点 3 是最右边的，第 2 层节点 4 是唯一的节点（2 的左子节点），从右侧可见。
```

## 约束条件

- 树中节点数目范围在 `[0, 100]` 内。
- `-100 <= Node.val <= 100`

## 函数签名

```go
func rightSideView(root *TreeNode) []int
```
