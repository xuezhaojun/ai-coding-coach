# 二叉树中的最大路径和

- **难度**: 困难
- **分类**: 树
- **考点**: 二叉树, 深度优先搜索, 树上动态规划
- **链接**: [NeetCode](https://neetcode.io/problems/binary-tree-maximum-path-sum) | [力扣 124](https://leetcode.cn/problems/binary-tree-maximum-path-sum/)

## 题目描述

给定一棵二叉树的根节点，返回其任意非空路径的最大路径和。二叉树中的路径是由一系列节点组成的序列，其中每对相邻节点之间存在一条边。每个节点在序列中最多出现一次。路径不需要经过根节点，可以从树中任意节点开始和结束。节点值可以为负数。

## 示例

**示例 1:**

```
输入: root = [1,2,3]
输出: 6
解释: 最优路径为 2 -> 1 -> 3，路径和为 2 + 1 + 3 = 6。
```

**示例 2:**

```
输入: root = [-10,9,20,null,null,15,7]
输出: 42
解释: 最优路径为 15 -> 20 -> 7，路径和为 15 + 20 + 7 = 42。
```

**示例 3:**

```
输入: root = [-3]
输出: -3
解释: 只有一个节点时，最大路径和就是该节点的值。
```

## 约束条件

- 树中节点数目范围在 `[1, 3 * 10^4]` 内。
- `-1000 <= Node.val <= 1000`

## 函数签名

```go
func maxPathSum(root *TreeNode) int
```
