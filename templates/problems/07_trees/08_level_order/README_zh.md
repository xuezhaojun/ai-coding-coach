# 二叉树的层序遍历

- **难度**: 中等
- **分类**: 树
- **考点**: 二叉树, 广度优先搜索, 队列
- **链接**: [NeetCode](https://neetcode.io/problems/level-order-traversal-of-binary-tree) | [力扣 102](https://leetcode.cn/problems/binary-tree-level-order-traversal/)

## 题目描述

给你二叉树的根节点，返回其节点值的层序遍历。层序遍历从左到右逐层处理节点，从根节点开始。结果是一个二维列表，其中每个内层列表包含对应深度层的所有节点值。

## 示例

**示例 1:**

```
输入: root = [3,9,20,null,null,15,7]
输出: [[3],[9,20],[15,7]]
解释: 第 0 层有节点 3，第 1 层有节点 9 和 20，第 2 层有节点 15 和 7。
```

**示例 2:**

```
输入: root = [1]
输出: [[1]]
解释: 树只有一个节点在第 0 层。
```

**示例 3:**

```
输入: root = []
输出: []
解释: 空树产生空结果。
```

## 约束条件

- 树中节点数目范围在 `[0, 2000]` 内。
- `-1000 <= Node.val <= 1000`

## 函数签名

```go
func levelOrder(root *TreeNode) [][]int
```
