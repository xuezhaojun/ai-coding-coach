# 翻转二叉树

- **难度**: 简单
- **分类**: 树
- **考点**: 二叉树, 递归, 深度优先搜索
- **链接**: [NeetCode](https://neetcode.io/problems/invert-a-binary-tree) | [力扣 226](https://leetcode.cn/problems/invert-binary-tree/)

## 题目描述

给定一棵二叉树的根节点，翻转这棵二叉树并返回其根节点。翻转二叉树意味着将每个节点的左子节点和右子节点互换，从而生成原始树的镜像。

## 示例

**示例 1:**

```
输入: root = [4,2,7,1,3,6,9]
输出: [4,7,2,9,6,3,1]
解释: 每对左右子节点都被交换。节点 2 和 7 交换，然后 1 和 3 交换，6 和 9 交换。
```

**示例 2:**

```
输入: root = [2,1,3]
输出: [2,3,1]
解释: 左子节点 1 和右子节点 3 被交换。
```

**示例 3:**

```
输入: root = []
输出: []
解释: 空树翻转后仍为空树。
```

## 约束条件

- 树中节点数目范围在 `[0, 100]` 内。
- `-100 <= Node.val <= 100`

## 函数签名

```go
func invertTree(root *TreeNode) *TreeNode
```
