# 买卖股票的最佳时机

- **难度**：简单
- **分类**：滑动窗口
- **考点**：数组、动态规划
- **链接**：[NeetCode](https://neetcode.io/problems/buy-and-sell-crypto) | [力扣 121](https://leetcode.cn/problems/best-time-to-buy-and-sell-stock/)

## 题目描述

给定一个数组 `prices`，它的第 `i` 个元素 `prices[i]` 表示一支给定股票第 `i` 天的价格。

你只能选择某一天买入这只股票，并选择在未来的某一个不同的日子卖出该股票。设计一个算法来计算你所能获取的最大利润。返回你可以从这笔交易中获取的最大利润。如果你不能获取任何利润，返回 `0`。

注意：你必须在买入之前卖出，并且最多只能完成一笔交易。

## 示例

**示例 1：**

```
输入：prices = [7,1,5,3,6,4]
输出：5
解释：在第 2 天（价格 = 1）买入，在第 5 天（价格 = 6）卖出，利润 = 6 - 1 = 5。注意不能在第 2 天买入后在第 1 天卖出，因为必须先买入再卖出。
```

**示例 2：**

```
输入：prices = [7,6,4,3,1]
输出：0
解释：在这种情况下，没有交易完成，最大利润为 0。
```

**示例 3：**

```
输入：prices = [1,2,3,4,5]
输出：4
解释：在第 1 天（价格 = 1）买入，在第 5 天（价格 = 5）卖出，利润 = 5 - 1 = 4。
```

## 约束条件

- `1 <= prices.length <= 10^5`
- `0 <= prices[i] <= 10^4`

## 函数签名

```go
func maxProfit(prices []int) int
```
