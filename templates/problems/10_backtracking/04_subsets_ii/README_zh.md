# 子集 II

- **难度**: 中等
- **分类**: 回溯
- **考点**: 回溯, 排序, 去重
- **链接**: [NeetCode](https://neetcode.io/problems/subsets-ii) | [力扣 90](https://leetcode.cn/problems/subsets-ii/)

## 题目描述

给你一个整数数组 `nums`，其中可能包含重复元素，请你返回该数组所有可能的子集（幂集）。解集不能包含重复的子集。返回的解集中，子集可以按任意顺序排列。

与基本子集问题不同，输入数组可能包含重复值。你必须确保输出中没有两个相同的子集。

## 示例

**示例 1:**

```
输入: nums = [1, 2, 2]
输出: [[], [1], [2], [1,2], [2,2], [1,2,2]]
```

**示例 2:**

```
输入: nums = [0]
输出: [[], [0]]
```

**示例 3:**

```
输入: nums = [1, 1, 1]
输出: [[], [1], [1,1], [1,1,1]]
```

## 约束条件

- `1 <= nums.length <= 10`
- `-10 <= nums[i] <= 10`

## 函数签名

```go
func subsetsWithDup(nums []int) [][]int
```
