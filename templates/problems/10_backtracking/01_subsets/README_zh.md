# 子集

- **难度**: 中等
- **分类**: 回溯
- **考点**: 回溯, 位运算, 递归
- **链接**: [NeetCode](https://neetcode.io/problems/subsets) | [力扣 78](https://leetcode.cn/problems/subsets/)

## 题目描述

给你一个整数数组 `nums`，数组中的元素互不相同。返回该数组所有可能的子集（幂集）。解集不能包含重复的子集，可以按任意顺序返回。

例如，给定 `nums = [1, 2, 3]`，输出应包含所有 8 个子集：空集、每个单独元素、每对元素的组合以及完整集合。

## 示例

**示例 1:**

```
输入: nums = [1, 2, 3]
输出: [[], [1], [2], [3], [1,2], [1,3], [2,3], [1,2,3]]
```

**示例 2:**

```
输入: nums = [0]
输出: [[], [0]]
```

**示例 3:**

```
输入: nums = [1, 2]
输出: [[], [1], [2], [1,2]]
```

## 约束条件

- `1 <= nums.length <= 10`
- `-10 <= nums[i] <= 10`
- 数组中的所有元素互不相同。

## 函数签名

```go
func subsets(nums []int) [][]int
```
