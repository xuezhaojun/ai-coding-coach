# 前 K 个高频元素

- **难度**：中等
- **分类**：数组与哈希
- **考点**：数组、哈希表、桶排序、堆
- **链接**：[NeetCode](https://neetcode.io/problems/top-k-elements-in-list) | [力扣 347](https://leetcode.cn/problems/top-k-frequent-elements/)

## 题目描述

给你一个整数数组 `nums` 和一个整数 `k`，请你返回其中出现频率前 `k` 高的元素。你可以按任意顺序返回答案。

题目数据保证答案唯一。

## 示例

**示例 1：**

```
输入：nums = [1,1,1,2,2,3], k = 2
输出：[1,2]
解释：1 出现 3 次，2 出现 2 次，3 出现 1 次。出现频率最高的两个元素是 1 和 2。
```

**示例 2：**

```
输入：nums = [1], k = 1
输出：[1]
```

**示例 3：**

```
输入：nums = [-1,-1,-2,-2,-2,-3], k = 1
输出：[-2]
解释：-2 出现 3 次，是出现频率最高的元素。
```

## 约束条件

- `1 <= nums.length <= 10^5`
- `-10^4 <= nums[i] <= 10^4`
- `k` 的取值范围是 `[1, 数组中不同元素的个数]`。
- 题目数据保证答案唯一。

## 函数签名

```go
func topKFrequent(nums []int, k int) []int
```
