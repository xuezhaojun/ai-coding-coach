# 寻找重复数

- **难度**：中等
- **分类**：链表
- **考点**：Floyd 判圈算法、双指针、数组
- **链接**：[NeetCode](https://neetcode.io/problems/find-the-duplicate-number) | [力扣 287](https://leetcode.cn/problems/find-the-duplicate-number/)

## 题目描述

给定一个包含 `n + 1` 个整数的数组 `nums`，其数字都在 `[1, n]` 范围内（包括 1 和 n），可知至少存在一个重复的整数。请你找出这个重复的数字。

你设计的解决方案必须不修改数组 `nums` 且只用常量级 O(1) 的额外空间。

## 示例

**示例 1：**

```
输入：nums = [1,3,4,2,2]
输出：2
解释：数字 2 在数组中出现了两次。
```

**示例 2：**

```
输入：nums = [3,1,3,4,2]
输出：3
解释：数字 3 在数组中出现了两次。
```

**示例 3：**

```
输入：nums = [1,1]
输出：1
解释：唯一可能的数字是 1，且它出现了两次。
```

## 约束条件

- `1 <= n <= 10^5`
- `nums.length == n + 1`
- `1 <= nums[i] <= n`
- `nums` 中只有一个整数出现两次或多次，其余整数均只出现一次

## 函数签名

```go
func findDuplicate(nums []int) int
```
