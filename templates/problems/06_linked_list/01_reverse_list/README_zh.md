# 反转链表

- **难度**：简单
- **分类**：链表
- **考点**：链表、迭代
- **链接**：[NeetCode](https://neetcode.io/problems/reverse-a-linked-list) | [力扣 206](https://leetcode.cn/problems/reverse-linked-list/)

## 题目描述

给你单链表的头节点 `head`，请你反转链表，并返回反转后的链表。

链表中的每个节点包含一个整数值和一个指向下一个节点的指针。反转后，最后一个节点变为第一个节点，所有指针方向反转，使链表以相反的方向遍历。

## 示例

**示例 1：**

```
输入：head = [1,2,3,4,5]
输出：[5,4,3,2,1]
解释：链表 1->2->3->4->5 变为 5->4->3->2->1。
```

**示例 2：**

```
输入：head = [1,2]
输出：[2,1]
解释：链表 1->2 变为 2->1。
```

**示例 3：**

```
输入：head = []
输出：[]
解释：空链表反转后仍为空。
```

## 约束条件

- 链表中节点的数目范围是 `[0, 5000]`
- `-5000 <= Node.Val <= 5000`

## 函数签名

```go
func reverseList(head *ListNode) *ListNode
```
