# 有效的括号

- **难度**：简单
- **分类**：栈
- **考点**：栈、字符串、哈希表
- **链接**：[NeetCode](https://neetcode.io/problems/validate-parentheses) | [力扣 20](https://leetcode.cn/problems/valid-parentheses/)

## 题目描述

给定一个只包括 `'('`、`')'`、`'{'`、`'}'`、`'['`、`']'` 的字符串 `s`，判断字符串是否有效。

有效字符串需满足：
1. 左括号必须用相同类型的右括号闭合。
2. 左括号必须以正确的顺序闭合。
3. 每个右括号都有一个对应的相同类型的左括号。

## 示例

**示例 1：**

```
输入：s = "()"
输出：true
```

**示例 2：**

```
输入：s = "()[]{}"
输出：true
```

**示例 3：**

```
输入：s = "(]"
输出：false
解释：左括号 '(' 与右括号 ']' 不匹配。
```

## 约束条件

- `0 <= s.length <= 10^4`
- `s` 仅由括号 `'()[]{}'` 组成

## 函数签名

```go
func isValid(s string) bool
```
