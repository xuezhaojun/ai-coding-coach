# 车队

- **难度**：中等
- **分类**：栈
- **考点**：栈、排序、单调栈
- **链接**：[NeetCode](https://neetcode.io/problems/car-fleet) | [力扣 853](https://leetcode.cn/problems/car-fleet/)

## 题目描述

在一条单行道上，有 `n` 辆车开往同一个目的地。目的地是距起点 `target` 英里处。给定两个整数数组 `position` 和 `speed`，长度都是 `n`，其中 `position[i]` 是第 `i` 辆车的出发位置，`speed[i]` 是第 `i` 辆车的速度（单位是英里/小时）。

一辆车永远不能超过前面的另一辆车，但它可以追上去，并以相同的速度紧跟着行驶。此时这两辆车被视为同一个车队。车队的速度是车队中最慢的一辆车的速度。注意，一辆车也可以视为一个车队。

如果一辆车恰好在目的地追上了一个车队，它们仍然被视为同一个车队。返回到达目的地的车队数量。

## 示例

**示例 1：**

```
输入：target = 12, position = [10,8,0,5,3], speed = [2,4,1,1,3]
输出：3
解释：从位置 10（速度 2）和位置 8（速度 4）出发的车会组成一个车队，在位置 12 相遇。从位置 0 出发的车无法追上任何车。从位置 5（速度 1）和位置 3（速度 3）出发的车会组成一个车队，在位置 6 相遇。所以共有 3 个车队。
```

**示例 2：**

```
输入：target = 10, position = [3], speed = [3]
输出：1
解释：只有一辆车，所以只有一个车队。
```

**示例 3：**

```
输入：target = 100, position = [0,50], speed = [1,1]
输出：2
解释：两辆车速度相同，但位置 50 的车在前方，它们永远不会相遇，形成 2 个独立的车队。
```

## 约束条件

- `n == position.length == speed.length`
- `1 <= n <= 10^5`
- `0 < target <= 10^6`
- `0 <= position[i] < target`
- `0 < speed[i] <= 10^6`
- `position` 中的所有值互不相同

## 函数签名

```go
func carFleet(target int, position []int, speed []int) int
```
