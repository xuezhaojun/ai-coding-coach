# 括号生成

- **难度**：中等
- **分类**：栈
- **考点**：回溯、字符串、递归
- **链接**：[NeetCode](https://neetcode.io/problems/generate-parentheses) | [力扣 22](https://leetcode.cn/problems/generate-parentheses/)

## 题目描述

数字 `n` 代表生成括号的对数，请你设计一个函数，用于能够生成所有可能的并且有效的括号组合。

有效括号组合需满足：每个左括号 `'('` 都有一个对应的右括号 `')'`，并且括号正确嵌套。

## 示例

**示例 1：**

```
输入：n = 1
输出：["()"]
```

**示例 2：**

```
输入：n = 2
输出：["(())","()()"]
```

**示例 3：**

```
输入：n = 3
输出：["((()))","(()())","(())()","()(())","()()()"]
解释：3 对括号共有 5 种有效的组合方式。
```

## 约束条件

- `1 <= n <= 8`

## 函数签名

```go
func generateParenthesis(n int) []string
```
