# 最大子数组和

- **难度**: 中等
- **分类**: 贪心
- **考点**: 数组, 动态规划, 贪心, Kadane 算法
- **链接**: [NeetCode](https://neetcode.io/problems/maximum-subarray) | [力扣 53](https://leetcode.cn/problems/maximum-subarray/)

## 题目描述

给你一个整数数组 `nums`，请你找出一个具有最大和的连续子数组（子数组最少包含一个元素），返回其最大和。子数组是数组中的一个连续非空元素序列。

## 示例

**示例 1:**

```
输入: nums = [-2,1,-3,4,-1,2,1,-5,4]
输出: 6
解释: 连续子数组 [4,-1,2,1] 的和最大，为 6。
```

**示例 2:**

```
输入: nums = [1]
输出: 1
解释: 子数组 [1] 的和最大，为 1。
```

**示例 3:**

```
输入: nums = [-1]
输出: -1
解释: 子数组 [-1] 的和最大，为 -1。即使是负数也必须选取至少一个元素。
```

## 约束条件

- `1 <= nums.length <= 10^5`
- `-10^4 <= nums[i] <= 10^4`

## 函数签名

```go
func maxSubArray(nums []int) int
```
