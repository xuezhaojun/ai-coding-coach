# 跳跃游戏 II

- **难度**: 中等
- **分类**: 贪心
- **考点**: 数组, 贪心, 广度优先搜索
- **链接**: [NeetCode](https://neetcode.io/problems/jump-game-ii) | [力扣 45](https://leetcode.cn/problems/jump-game-ii/)

## 题目描述

给定一个长度为 `n` 的 0 索引整数数组 `nums`。初始位置为 `nums[0]`。每个元素 `nums[i]` 表示从索引 `i` 向前跳转的最大长度。返回到达 `nums[n - 1]` 的最小跳跃次数。生成的测试用例可以到达 `nums[n - 1]`。

## 示例

**示例 1:**

```
输入: nums = [2,3,1,1,4]
输出: 2
解释: 跳到最后一个位置的最小跳跃数是 2。从下标 0 跳 1 步到下标 1，然后跳 3 步到达最后一个下标。
```

**示例 2:**

```
输入: nums = [2,3,0,1,4]
输出: 2
```

**示例 3:**

```
输入: nums = [0]
输出: 0
解释: 已经在最后一个下标，不需要跳跃。
```

## 约束条件

- `1 <= nums.length <= 10^4`
- `0 <= nums[i] <= 1000`
- 题目保证可以到达 `nums[n - 1]`。

## 函数签名

```go
func jump(nums []int) int
```
