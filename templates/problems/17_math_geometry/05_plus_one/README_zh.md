# 加一

- **难度**: 简单
- **分类**: 数学与几何
- **考点**: 数组, 数学
- **链接**: [NeetCode](https://neetcode.io/problems/plus-one) | [力扣 66](https://leetcode.cn/problems/plus-one/)

## 题目描述

给定一个由整数组成的非空数组 `digits` 所表示的非负整数，其中每个 `digits[i]` 是该整数的第 `i` 位数字。数字按从最高有效位到最低有效位的顺序从左到右排列，且该大整数不含任何前导 `0`。

在该数的基础上加一，返回结果对应的数字数组。

## 示例

**示例 1:**

```
输入: digits = [1,2,3]
输出: [1,2,4]
解释: 数组表示整数 123，加一后为 124。
```

**示例 2:**

```
输入: digits = [4,3,2,9]
输出: [4,3,3,0]
解释: 数组表示整数 4329，加一后为 4330。
```

**示例 3:**

```
输入: digits = [9,9,9]
输出: [1,0,0,0]
解释: 数组表示整数 999，加一后为 1000，需要额外一位。
```

## 约束条件

- `1 <= digits.length <= 100`
- `0 <= digits[i] <= 9`
- `digits` 不包含任何前导 `0`

## 函数签名

```go
func plusOne(digits []int) []int
```
