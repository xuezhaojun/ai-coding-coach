# 矩阵置零

- **难度**: 中等
- **分类**: 数学与几何
- **考点**: 数组, 矩阵, 哈希表
- **链接**: [NeetCode](https://neetcode.io/problems/set-zeroes-in-matrix) | [力扣 73](https://leetcode.cn/problems/set-matrix-zeroes/)

## 题目描述

给定一个 `m x n` 的整数矩阵 `matrix`，如果一个元素为 `0`，则将其所在行和列的所有元素都设为 `0`。请使用原地算法，不要使用与矩阵大小成比例的额外空间。

## 示例

**示例 1:**


![示例 1](assets/mat1.jpg)
```
输入: matrix = [[1,1,1],[1,0,1],[1,1,1]]
输出: [[1,0,1],[0,0,0],[1,0,1]]
解释: 位置 (1,1) 的元素为 0，因此其所在行和列全部置零。
```

**示例 2:**


![示例 2](assets/mat2.jpg)
```
输入: matrix = [[0,1,2,0],[3,4,5,2],[1,3,1,5]]
输出: [[0,0,0,0],[0,4,5,0],[0,3,1,0]]
解释: 位置 (0,0) 和 (0,3) 的元素为 0，其所在行和列全部置零。
```

**示例 3:**

```
输入: matrix = [[1,2],[3,4]]
输出: [[1,2],[3,4]]
解释: 没有元素为 0，矩阵保持不变。
```

## 约束条件

- `m == matrix.length`
- `n == matrix[0].length`
- `1 <= m, n <= 200`
- `-2^31 <= matrix[i][j] <= 2^31 - 1`

## 函数签名

```go
func setZeroes(matrix [][]int)
```
