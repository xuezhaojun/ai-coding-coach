# Pow(x, n)

- **难度**: 中等
- **分类**: 数学与几何
- **考点**: 数学, 递归, 快速幂
- **链接**: [NeetCode](https://neetcode.io/problems/pow-x-n) | [力扣 50](https://leetcode.cn/problems/powx-n/)

## 题目描述

实现 `pow(x, n)`，即计算 `x` 的 `n` 次幂（即 `x^n`）。函数需要处理负指数的情况（例如 `x^(-n) = 1 / x^n`），且不能简单地循环 `n` 次，因为 `n` 可能非常大。

## 示例

**示例 1:**

```
输入: x = 2.00000, n = 10
输出: 1024.00000
解释: 2^10 = 1024。
```

**示例 2:**

```
输入: x = 2.00000, n = -2
输出: 0.25000
解释: 2^(-2) = 1 / 2^2 = 1/4 = 0.25。
```

**示例 3:**

```
输入: x = 2.10000, n = 3
输出: 9.26100
解释: 2.1^3 = 9.261。
```

## 约束条件

- `-100.0 < x < 100.0`
- `-2^31 <= n <= 2^31 - 1`
- `n` 是整数
- `x` 不为零或 `n > 0`
- `-10^4 <= x^n <= 10^4`

## 函数签名

```go
func myPow(x float64, n int) float64
```
