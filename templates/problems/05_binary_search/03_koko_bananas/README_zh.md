# 爱吃香蕉的珂珂

- **难度**：中等
- **分类**：二分查找
- **考点**：二分查找、数学
- **链接**：[NeetCode](https://neetcode.io/problems/eating-bananas) | [力扣 875](https://leetcode.cn/problems/koko-eating-bananas/)

## 题目描述

珂珂喜欢吃香蕉。这里有 `n` 堆香蕉，第 `i` 堆中有 `piles[i]` 根香蕉。警卫已经离开了，将在 `h` 小时后回来。

珂珂可以决定她吃香蕉的速度 `k`（单位：根/小时）。每个小时，她将会选择一堆香蕉，从中吃掉 `k` 根。如果这堆香蕉少于 `k` 根，她将吃掉这堆的所有香蕉，然后这一小时内不会再吃更多的香蕉。

珂珂喜欢慢慢吃，但仍然想在警卫回来之前吃掉所有的香蕉。返回她可以在 `h` 小时内吃掉所有香蕉的最小速度 `k`（`k` 为整数）。

## 示例

**示例 1：**

```
输入：piles = [3,6,7,11], h = 8
输出：4
解释：速度为 4 时，珂珂需要 1+2+2+3 = 8 小时，刚好在 h = 8 小时内完成。
```

**示例 2：**

```
输入：piles = [30,11,23,4,20], h = 5
输出：30
解释：5 堆香蕉只有 5 小时，珂珂必须每小时吃完一堆，所以速度至少为 max(piles) = 30。
```

**示例 3：**

```
输入：piles = [30,11,23,4,20], h = 6
输出：23
解释：速度为 23 时，珂珂需要 2+1+1+1+1 = 6 小时，在 h = 6 小时内完成。
```

## 约束条件

- `1 <= piles.length <= 10^4`
- `piles.length <= h <= 10^9`
- `1 <= piles[i] <= 10^9`

## 函数签名

```go
func minEatingSpeed(piles []int, h int) int
```
